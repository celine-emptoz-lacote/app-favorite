Autres fonctionnalités

Choisissez ce que vous partagez

Avec Connexion Facebook, vous choisissez quelles applications peuvent partager avec vos amis. Vous pouvez aussi « ignorer » entièrement le partage.

Connectez-vous plus rapidement aux applications

Connectez-vous à vos applications favorites en quelques touches. Plus besoin de remplir de longs formulaires ou de vous souvenir d’un nom d’utilisateur ou d’un mot de passe.

Utilisé par les applications que vous aimez

La fonctionnalité Connexion Facebook est utilisée par les meilleures applications sur iOS et Android. Nous examinons les applications pour nous assurer qu’elles proposent de bonnes expériences de connexion.
